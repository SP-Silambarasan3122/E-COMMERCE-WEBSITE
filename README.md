E-Commerce Website

A responsive E-Commerce website developed using HTML, CSS, and JavaScript.  
This project simulates a basic online shopping platform where users can browse products, search items, add them to a cart, and complete the checkout process.

- Project Features

✔ Home page with navigation bar and image slider  
✔ Product listing with product cards  
✔ Product search functionality  
✔ Category filtering system  
✔ Add to cart functionality  
✔ Shopping cart page with total price calculation  
✔ Checkout page with shipping form  
✔ Order confirmation page  
✔ Contact form validation using JavaScript  
✔ Responsive design for mobile and desktop  

- Technologies Used

- HTML5  
- CSS3  
- JavaScript (ES6)  
- LocalStorage (for cart system)  
- JSON (for product data)
